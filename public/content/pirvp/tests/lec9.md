# Тест: Асинхронность в JavaScript

## question
Тип: single
Вопрос: Что такое Promise?
- Синхронная операция
- Объект представляющий завершение асинхронной операции
- Функция
- Массив
Ответ: Объект представляющий завершение асинхронной операции
Пояснение: Promise представляет результат асинхронной операции.

## question
Тип: multiple
Вопрос: Какие методы есть у Promise?
- then()
- catch()
- finally()
- await()
Ответ: then(), catch(), finally()
Пояснение: await — ключевое слово, не метод Promise.

## question
Тип: single
Вопрос: Что такое Event Loop?
- Массив событий
- Механизм обработки очереди событий в JavaScript
- Функция обработки кликов
- Метод объекта
Ответ: Механизм обработки очереди событий в JavaScript
Пояснение: Event Loop отвечает за выполнение кода в правильном порядке.
