# Тест: Основы CSS

## question
Тип: single
Вопрос: Какое свойство отвечает за внешние отступы?
- padding
- margin
- border
- gap
Ответ: margin
Пояснение: margin — внешние отступы элемента от других элементов.

## question
Тип: multiple
Вопрос: Какие способы задания цвета существуют в CSS?
- Имя цвета
- HEX
- RGB
- JSON
Ответ: Имя цвета, HEX, RGB
Пояснение: JSON не используется для задания цвета в CSS.
