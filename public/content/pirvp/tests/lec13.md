# Тест: Основы React

## question
Тип: single
Вопрос: Что такое JSX?
- JavaScript XML
- Java Syntax Extension
- JSON XML
- Java XML
Ответ: JavaScript XML
Пояснение: JSX — расширение синтаксиса JavaScript для описания UI.

## question
Тип: single
Вопрос: Что такое state в React?
- Глобальная переменная
- Данные компонента, которые могут изменяться
- Стили компонента
- Серверные данные
Ответ: Данные компонента, которые могут изменяться
Пояснение: State — это данные, которые хранятся внутри компонента.

## question
Тип: multiple
Вопрос: Какие хуки есть в React?
- useState
- useEffect
- useRouter
- useMemo
Ответ: useState, useEffect, useMemo
Пояснение: useRouter — не стандартный хук React.
