# Тест: Работа браузера и семантический HTML

## question
Тип: single
Вопрос: Что такое DOM?
- База данных
- Объектная модель документа
- Язык разметки
- Протокол передачи данных
Ответ: Объектная модель документа
Пояснение: DOM (Document Object Model) — программный интерфейс документа.

## question
Тип: multiple
Вопрос: Какие теги являются семантическими?
- div
- article
- nav
- header
Ответ: article, nav, header
Пояснение: Семантические теги описывают смысл контента.
