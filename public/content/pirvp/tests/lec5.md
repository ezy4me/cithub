# Тест: Основы JavaScript

## question
Тип: single
Вопрос: Какой метод добавляет элемент в конец массива?
- push()
- pop()
- shift()
- unshift()
Ответ: push()
Пояснение: push() добавляет элемент в конец массива.

## question
Тип: multiple
Вопрос: Какие из этих способов создают функцию?
- function f() {}
- () => {}
- new Function()
- createFunc()
Ответ: function f() {}, () => {}, new Function()
Пояснение: Все три способа создают функцию.
