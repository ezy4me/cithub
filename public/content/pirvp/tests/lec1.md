# Тест: Введение в веб-технологии

## question
Тип: single
Вопрос: Какой протокол используется для передачи веб-страниц?
- HTTP
- FTP
- SMTP
- SSH
Ответ: HTTP
Пояснение: HTTP (HyperText Transfer Protocol) — протокол передачи гипертекста.

## question
Тип: single
Вопрос: Что такое HTML?
- Язык программирования
- Язык разметки гипертекста
- База данных
- Сервер
Ответ: Язык разметки гипертекста
Пояснение: HTML расшифровывается как HyperText Markup Language.

## question
Тип: multiple
Вопрос: Какие из перечисленных являются клиентскими языками?
- JavaScript
- Python
- CSS
- PHP
Ответ: JavaScript, CSS
Пояснение: JavaScript и CSS выполняются в браузере клиента.
