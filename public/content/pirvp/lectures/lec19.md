# Лекция №19. SEO

## 1. Введение в SEO

### 1.1. Что такое SEO

SEO (Search Engine Optimization, поисковая оптимизация) — это набор методов и практик, направленных на улучшение видимости сайта в поисковых системах, таких как Google, Яндекс или Bing.

Главная цель SEO — чтобы сайт появлялся как можно выше в органической выдаче по релевантным запросам. Органическая выдача отличается от рекламы: это бесплатные результаты поиска, которые поисковые системы считают наиболее полезными для пользователя.

**Простой пример:** если пользователь ищет «лучший интернет-магазин книг», то хорошо оптимизированный сайт магазина книг будет отображаться выше конкурентов, если SEO выполнено правильно.

**Особенность для разработчиков:** тип рендеринга страниц влияет на SEO.  
- **CSR (Client-Side Rendering)** — контент рендерится в браузере через JavaScript. Поисковые роботы иногда не видят сразу весь контент, что может снизить эффективность SEO.  
- **SSR (Server-Side Rendering)** — контент генерируется на сервере и сразу отправляется в HTML. Поисковые системы видят контент сразу, что улучшает индексацию и ранжирование.

### 1.2. Зачем SEO важно для веб-сайта

SEO важно потому, что:  
- Большинство пользователей кликают на первые 5–10 результатов поиска.  
- Хорошее SEO повышает трафик без необходимости платить за рекламу.  
- Пользователи доверяют органическим результатам больше, чем платной рекламе.  
- SEO помогает улучшить юзабилити: структурированный сайт, быстрая загрузка и понятная навигация положительно влияют на опыт пользователей.

**Пример:** магазин с хорошей SEO-оптимизацией получает на 50–70% больше органических посетителей, чем сайт без оптимизации.

### 1.3. Основные цели SEO

Основные цели SEO можно разделить на три группы:

1. **Повышение видимости в поиске**  

- Чтобы сайт показывался по большему числу релевантных запросов.  
- Например, интернет-магазин одежды хочет появляться по запросам «женские платья», «мужские рубашки», «детская одежда».

2. **Привлечение целевого трафика**  

- Важно не просто получать посетителей, а чтобы они были заинтересованы в продукте или услуге.  
- Пример: сайт фотостудии привлекает людей, которые ищут «семейная фотосессия в Москве», а не просто «фото».

3. **Улучшение конверсии и пользовательского опыта**  

- SEO включает техническую оптимизацию: скорость загрузки, адаптивный дизайн, удобная структура.  
- Это помогает пользователям легче находить информацию, а владельцу сайта — конвертировать посетителей в клиентов.

## 2. Ключевые элементы SEO на странице

On-page SEO — это совокупность факторов на самой странице сайта, которые влияют на её видимость в поисковых системах. Цель — сделать страницу удобной для пользователей и понятной для поисковых роботов, чтобы повысить позиции в выдаче.

### 2.1. Title и Meta Description

**Title** — это заголовок страницы, который виден в поисковой выдаче и браузере. Он должен быть:  
- уникальным,  
- включать ключевое слово,  
- длиной до 60 символов.

**Meta Description** — краткое описание страницы под заголовком в выдаче. Рекомендуется:  
- привлекать внимание,  
- содержать ключевые слова,  
- быть длиной до 160 символов.


### 2.2. Заголовки H1–H6

Заголовки структурируют контент:  
- **H1** отражает тему страницы, один на страницу,  
- **H2–H3** делят текст на логические блоки, делают контент удобным для чтения и понятным для поисковых систем.

**Пример:**
```html
<h1>Рецепты домашних тортов</h1>
<h2>Шоколадные торты</h2>
<h3>Шоколадный бисквит</h3>
<h2>Фруктовые торты</h2>
```

### 2.3. URL страницы

URL должен быть:  
- коротким и понятным,  
- содержать ключевые слова,  
- использовать дефисы для разделения слов,  
- избегать лишних символов и цифр.

**Пример:**  
`https://www.sweethome.ru/retsepty-domashnikh-tortov`

### 2.4. Контент и ключевые слова

Контент должен быть:  
- уникальным и полезным,  
- структурированным для удобства восприятия,  
- с естественным использованием ключевых слов.

**Пример (текст статьи о тортах):**  
«Если вы ищете лучшие рецепты домашних тортов, начните с классического шоколадного бисквита. Наши пошаговые инструкции помогут вам приготовить вкусный десерт даже новичку.»

Для улучшения восприятия можно использовать:  
- списки,  
- таблицы,  
- изображения.

### 2.5. Внутренние ссылки

Ссылки между страницами сайта помогают поисковым системам индексировать контент и распределять «вес» страниц. Рекомендуется:  
- вести на связанные материалы,  
- использовать анкор с ключевыми словами,  
- не перегружать страницу ссылками.

**Пример:**
```html
<p>Попробуйте также <a href="/shokoladnye-torty">рецепты шоколадных тортов</a> для разнообразия вашего меню.</p>
```

### 2.6. Мобильная оптимизация и адаптивный дизайн

- Страницы должны корректно отображаться на всех устройствах.  
- Google использует mobile-first indexing — сначала анализируется мобильная версия сайта.

**Пример CSS:**
```css
@media (max-width: 768px) {
  .menu { display: none; }
}
```

## 3. Контент и ключевые слова

Контент — один из самых важных факторов SEO. Поисковые системы оценивают страницы по качеству текста, мультимедиа и удобству восприятия информации. Хороший контент помогает привлекать целевых пользователей и повышает конверсию.

### 3.1. Исследование ключевых слов

Исследование ключевых слов (keyword research) помогает понять, какие запросы используют пользователи, чтобы найти ваш продукт или услугу.

**Рекомендации:**  
- Использовать специализированные сервисы: Google Keyword Planner, Яндекс.Вордстат, SEMrush, Ahrefs.  
- Определять частотность и конкуренцию ключевых слов.  
- Делить слова на категории: информационные, коммерческие, навигационные.

**Для сайта пекарни ключевые слова могут быть:**  
- «рецепты домашних тортов» (информационный запрос)  
- «купить шоколадный торт в Москве» (коммерческий запрос)  
- «пекарня SweetHome» (навигационный запрос)

### 3.2. Семантическое ядро и плотность ключевых слов

Семантическое ядро — это набор ключевых слов и фраз, которые используются на сайте для охвата целевой аудитории.

**Рекомендации:**  
- Ключевые слова должны встречаться естественно в тексте.  
- Оптимальная плотность — 1–3% для основного ключа.  
- Использовать синонимы и LSI-слова (словосочетания с похожим смыслом).

**Пример (Текст статьи о тортах):**  
«Если вы ищете лучшие рецепты домашних тортов, начните с классического шоколадного бисквита. Наши пошаговые инструкции помогут приготовить вкусный десерт даже новичку.»

**Ключевые слова:** «рецепты домашних тортов», «шоколадный бисквит», «пошаговые инструкции».

### 3.3. Контент-маркетинг и уникальный текст

Контент должен быть:  
- Уникальным, а не скопированным с других сайтов.  
- Полезным для пользователей: инструкции, советы, кейсы, инфографика.  
- Структурированным: заголовки, списки, таблицы, изображения.

**Пример:**  
- Статья на сайте пекарни: пошаговый рецепт с фото, таблицей ингредиентов и советами по выпеканию.  
- Блог магазина: обзоры сезонных новинок с рекомендациями и видео-уроками.

**Дополнительно:**  
- Видео и инфографика повышают время пребывания на сайте.  
- Четкая структура помогает поисковикам индексировать страницы и улучшает UX.

## 4. Техническое SEO

Техническое SEO отвечает за «инфраструктуру» сайта — как поисковые системы видят и индексируют страницы. Хорошая техническая оптимизация повышает позиции сайта и улучшает пользовательский опыт.

### 4.1. Скорость загрузки сайта

Скорость загрузки влияет на поведение пользователей и на ранжирование в поисковых системах.

**Рекомендации:**  
- Минимизировать HTML, CSS и JavaScript.  
- Использовать сжатие изображений (WebP, JPEG 2000).  
- Включить кэширование и CDN (Content Delivery Network).  
- Оптимизировать серверные настройки.

**Пример:**  
- Страница с тяжелыми изображениями и множеством скриптов может загружаться 8–10 секунд → высокий показатель отказов.  
- Оптимизированная страница загружается за 2–3 секунды → лучше ранжируется и удерживает пользователей.

### 4.2. Sitemap и robots.txt

**Sitemap** — это файл с перечнем всех страниц сайта, который помогает поисковым системам индексировать контент.  
**Robots.txt** — файл, который управляет доступом поисковых ботов к страницам.

**Рекомендации:**  
- Sitemap должен содержать актуальные URL и быть доступен по адресу `/sitemap.xml`.  
- Robots.txt блокирует лишние страницы (например, админ-панель, временные страницы).

**Пример:**  
Sitemap: `https://www.sweethome.ru/sitemap.xml`  
```
User-agent: *
Disallow: /admin/
```

### 4.3. Канонические URL и дублированный контент

Канонические URL (`rel="canonical"`) помогают указать поисковикам основную версию страницы, если есть дубли.

**Рекомендации:**  
- Использовать канонические ссылки на страницах с похожим контентом.  
- Избегать дублей с разными URL, например:  
  `https://www.sweethome.ru/tort` и `https://sweethome.ru/tort/`

**Пример:**
```html
<link rel="canonical" href="https://www.sweethome.ru/tort">
```

### 4.4. HTTPS и безопасность

HTTPS — обязательный стандарт для современных сайтов:  
- Обеспечивает защищённое соединение.  
- Улучшает доверие пользователей.  
- Является фактором ранжирования в Google.

**Пример:**  
- Старый HTTP-сайт → браузеры показывают предупреждение «не защищено».  
- HTTPS-сайт → безопасный, с зелёным замком в адресной строке → лучше конверсия и SEO.

## 5. Внешнее SEO и ссылки

Внешнее SEO (Off-page SEO) — это действия за пределами сайта, которые влияют на его авторитет и позиции в поисковой выдаче. Главный фактор — ссылки с других сайтов и взаимодействие с пользователями.

### 5.1. Внутренние ссылки

Внутренние ссылки помогают поисковым системам понимать структуру сайта и распределять «вес» страниц.

**Рекомендации:**  
- Вести на тематически связанные страницы.  
- Использовать анкор с ключевыми словами.  
- Не перегружать страницы лишними ссылками.

**Пример:**
```html
<p>Попробуйте также <a href="/shokoladnye-torty">рецепты шоколадных тортов</a> для разнообразия вашего меню.</p>
```

### 5.2. Внешние ссылки (backlinks)

Backlinks — ссылки на ваш сайт с других ресурсов. Они повышают доверие поисковых систем и авторитет сайта.

**Рекомендации:**  
- Стараться получать ссылки с тематических и авторитетных ресурсов.  
- Избегать спамных ссылок с низкокачественных сайтов.  
- Разнообразить анкор-тексты: использовать брендовые, ключевые и естественные формулировки.

**Пример:**  
- Кулинарный блог ссылается на ваш рецепт «Шоколадный торт» → повышается доверие Google.

### 5.3. Социальные сигналы и упоминания

Социальные сети и упоминания бренда косвенно влияют на SEO:  
- Увеличивают трафик и вовлечённость.  
- Повышают узнаваемость бренда.  
- Ссылки из социальных сетей не всегда передают «вес», но помогают индексации и репутации.

**Пример:**  
- Пост в Telegram с ссылкой на рецепт → люди переходят на сайт → увеличивается органический трафик.  
- Обзор сайта на тематическом форуме → пользователи делятся ссылкой → появляется естественный backlink.

## 6. Аналитика: Google Analytics и Яндекс.Метрика

Для понимания поведения пользователей и эффективности сайта используют разные инструменты аналитики:

**Google Analytics:**  
- Основной инструмент для анализа веб-трафика.  
- Помогает видеть, откуда приходят пользователи, какие страницы популярны, какие действия выполняются на сайте.  
- Используется для оценки конверсий и улучшения маркетинговых стратегий.

**Яндекс.Метрика:**  
- Русскоязычный аналог GA, ориентирован на сайты для аудитории России и СНГ.  
- Встроенные карты кликов, скролла и форм помогают улучшить удобство сайта.  
- Позволяет отслеживать источники трафика и конверсии.

**Кратко:** оба сервиса помогают:  
- Анализировать поведение пользователей;  
- Определять слабые места сайта;  
- Улучшать конверсии и юзабилити;  
- Планировать маркетинговые кампании.

**Пример использования:** «Посмотрели карту кликов → заметили, что кнопка “Купить” плохо видна → изменили дизайн → повысили конверсии».
