# Тест: Введение в Git

## question
Тип: single
Вопрос: Какая команда создаёт новый коммит?
- git add
- git commit
- git push
- git pull
Ответ: git commit
Пояснение: git commit создаёт коммит с изменениями.

## question
Тип: multiple
Вопрос: Какие команды добавляют файл в staging?
- git add
- git commit
- git stage
- git init
Ответ: git add, git stage
Пояснение: git add и git stage добавляют файлы в staging.
