# Тест: Работа с удалённым репозиторием

## question
Тип: single
Вопрос: Какая команда клонирует репозиторий?
- git clone
- git copy
- git download
- git fetch
Ответ: git clone
Пояснение: git clone создаёт копию удалённого репозитория.

## question
Тип: single
Вопрос: Что делает git merge?
- Удаляет ветку
- Сливает изменения из одной ветки в другую
- Создаёт новую ветку
- Отправляет изменения
Ответ: Сливает изменения из одной ветки в другую
Пояснение: git merge объединяет историю коммитов двух веток.
