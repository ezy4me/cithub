# Cithub

Образовательная платформа для студентов-программистов от **MRS Education**.

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен сборки
npm run preview
```

## 🛠 Технологии

- **React 18** + TypeScript
- **Vite** — быстрая сборка
- **Tailwind CSS v4** — стилизация
- **shadcn/ui** — компоненты
- **React Router** — навигация
- **Zustand** — управление состоянием
- **Framer Motion** — анимации
- **React Markdown** — рендеринг Markdown

## 📁 Структура проекта

```
src/
├── app/              # Основные настройки приложения
├── pages/            # Страницы приложения
├── components/
│   ├── layout/       # Layout компоненты (Sidebar, BottomNav)
│   └── ui/          # Базовые UI компоненты
├── entities/         # Бизнес-сущности (Subject, Lecture, Quiz)
├── features/         # Фичи (quiz)
├── shared/
│   ├── hooks/       # React хуки
│   ├── lib/         # Утилиты и stores
│   └── components/  # Переиспользуемые компоненты
├── api/              # API слой (для масштабирования)
└── data/            # Mock данные

public/
├── config/          # Конфигурация предметов и лекций
└── materials/       # Markdown файлы лекций
```

## 🎨 Особенности

### Навигация
- **Desktop**: Sidebar с навигацией, поиском и списком предметов
- **Mobile**: Bottom Navigation

### Страницы
- `/` — Главная (продолжить обучение, прогресс, избранное)
- `/subjects` — Все предметы (grid карточек)
- `/subject/:id` — Страница предмета (лекции, практики, лабораторные)
- `/lecture/:id` — Страница лекции с Markdown контентом
- `/lecture/:id/presentation` — Режим презентации (слайды)
- `/search` — Поиск по предметам и лекциям
- `/favorites` — Избранные лекции
- `/settings` — Расширенные настройки

### Настройки
- Темы: светлая, тёмная, системная
- Акцентные цвета: чёрный, фиолетовый, зелёный, оранжевый, розовый, красный
- Шрифты: Sans, Mono, Serif
- Размер шрифта и межстрочный интервал
- Анимации и режим фокуса
- Стиль блоков кода

### Управление состоянием
- **Zustand stores** с persist в localStorage:
  - `settingsStore` — тема, акцентный цвет, шрифт, анимации
  - `favoritesStore` — избранные лекции
  - `progressStore` — прогресс обучения

## 📝 Markdown

Лекции поддерживают:
- Заголовки (h1-h6)
- Списки (ul, ol)
- Таблицы
- Блоки кода с подсветкой синтаксиса
- Кнопка копирования кода
- Отображение языка программирования
- Цитаты
- Изображения
- Разделители `---` для презентаций

## 🎞 Презентации

Markdown → слайды (разделение `---`):
- Шапка с информацией о лекции
- Счётчик слайдов (01/12)
- Прогресс-бар (в процентах)
- Навигация: стрелки, клики, клавиши
- Fullscreen режим (F)
- Точечный индикатор прогресса

## 🔌 Конфигурация

Материалы хранятся в отдельных файлах:

```
public/config/subjects.json   # Метаданные предметов
public/config/lectures.json   # Метаданные лекций
public/content/    # Markdown файлы
```
